## [First Human Uterus Maintained Outside Body in Medical Breakthrough](https://www.technologyreview.com/2026/03/28/1134766/womans-uterus-kept-alive-outside-the-body-first/) ⭐️ 8.0/10

Researchers have successfully maintained a woman's uterus outside her body for the first time using a specialized perfusion system that mimics natural blood supply through plastic tubing, developed by a team led by Javier González. This breakthrough represents a significant advancement in reproductive medicine with potential applications for fertility treatments and uterus transplants, offering new hope for women with uterine factor infertility. The perfusion system uses flexible plastic tubing as veins and arteries to continuously pump blood through the organ, mimicking natural circulation and preventing tissue damage that occurs during cold storage.

rss · MIT Technology Review · Mar 28, 09:00

**Background**: Ex vivo organ perfusion is a technique that uses machines to keep organs warm by continuously pumping blood through them, also called normothermic perfusion. This approach significantly reduces cold ischemic time, which is the time an organ spends on ice without blood flow. While this technology has been used for hearts, lungs, kidneys, and livers, applying it to the uterus presents unique challenges due to the organ's complex vascular structure.

## [Show HN: Pneuma - A Pure AI-Generated Desktop OS](https://pneuma.computer/) ⭐️ 7.0/10

Pneuma is an experimental desktop OS where users boot to a blank screen and describe what application they need; an LLM then generates a self-contained Rust module, compiles it to WebAssembly in under a second, and runs it in a sandboxed Wasmtime instance with GPU rendering via wgpu. This represents a paradigm shift from the traditional 'find, install, learn, configure' software model to instant on-demand tool generation. It demonstrates that AI can create functional software in real-time, potentially eliminating the need for pre-built application libraries. The system achieves ~90% first-attempt success rate, with automatic error feedback for correction. Agents run in isolated WASM sandboxes with a typed ABI for drawing, input, storage, and networking—agent crashes cannot bring down the system. Currently supports Linux, macOS, and Windows desktop apps.

rss · Hacker News - Show HN · Mar 28, 18:38

**Background**: wgpu is a safe and portable Rust graphics library based on the WebGPU API, supporting Vulkan, Metal, DirectX 12, and OpenGL ES backends. Wasmtime is a lightweight WebAssembly runtime developed by the Bytecode Alliance that provides fast, secure, and standards-compliant JIT compilation and sandboxed execution. The combination enables cross-platform GPU-accelerated applications running in secure sandboxes.
